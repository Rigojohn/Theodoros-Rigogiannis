# Do Expected Stock Returns Really Exist?

This project explores whether expected stock returns can be treated as stable and measurable quantities in financial markets.

The analysis combines concepts from probability theory, stochastic processes, ergodicity, Monte Carlo simulation and financial time series modeling. The main idea is to examine why historical averages may be misleading when returns are non-stationary, path-dependent or non-ergodic.

## Objective

The goal of this project is to challenge the traditional use of expected returns in financial models and show that estimated returns can change significantly depending on the sample period, time horizon and underlying market regime.

## Methods

- Monte Carlo simulations
- Law of Large Numbers and convergence analysis
- Ergodic vs non-ergodic processes
- NVIDIA daily return analysis
- Cumulative mean return comparison
- Discussion of Kalman Filtering, regime-switching models and GARCH-in-Mean

## Key Takeaway

Expected returns should not always be viewed as fixed parameters. In real financial markets, they may behave more like time-varying stochastic processes, requiring more adaptive modeling approaches.
